Buttons

Main styles

button.primary (full background) or button.outline (only border) add .disabled for inactive buttons

Additional styles

.rounded - 5px border-radius
.radial - 50px border-radius

Hovers

.hover-lighten - for .primary; drops the alpha of bg to .75
.hover-full - for .outline. Takes the color of the border and sets it as the bg

Text color

By default is set automatically by checking the bg color. If the bg is dark, gives light text and vice versa 
                        
<button class="col sm_3 md_3 lg_ xlg_4 primary deepblue-100 rounded">primary rounded</button>
<button class="col sm_3 md_3 lg_ xlg_4 outline deepblue-100 radial ">outline radial</button>
<button class="sm_3 md_3 lg_ xlg_4 outline deepblue-100 col ">outline</button>

Inputs

error-message=""; Can display the error message using .floating-label:after. Need to add .error to .floating-label to display the error message and change bottom-border to $red;

id of the input has to be the same as label for=""

<script>$.swInputRange();</script>  // For input range
